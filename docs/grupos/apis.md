# APIs e automação

Este grupo cobre as capacidades de **integração por API REST**: documentação, autenticação, automação de rotinas e uso de dashboards/relatórios por sistemas externos.

## Itens e explicação

- **48.** Disponibilizar APIs REST documentadas e autenticadas para integração com sistemas internos e externos da Câmara Municipal.  
  - **Explicação**: APIs com documentação clara e autenticação (tokens/chaves) para consumo seguro por outros sistemas.

- **49.** Permitir integração via API REST de dashboards, com parâmetros e tokens de autenticação.  
  - **Explicação**: permitir incorporar/consultar dashboards por API, com parâmetros (filtros) e segurança (token).

- **50.** Permitir integração via API REST de usuários, com funções para criar, listar, editar, inativar e gerenciar grupos, permissões e unidades administrativas.  
  - **Explicação**: governança de identidade via API (provisionamento e administração automatizada).

- **51.** Permitir integração via API REST para geração de relatórios sob demanda e definição de formatos de saída (PDF, XLS, HTML).  
  - **Explicação**: solicitar relatórios parametrizados de forma programática e receber em formatos padronizados.

- **52.** Permitir integração via API REST para criação, ativação, desativação e listagem de agendamentos automáticos.  
  - **Explicação**: automatizar agendamentos (ciclo de vida) para distribuição recorrente sem operação manual.
